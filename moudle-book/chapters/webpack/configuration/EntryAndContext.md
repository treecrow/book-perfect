# [Entry and Context](https://webpack.js.org/configuration/entry-context/)

| option  | more                                                                                                     |
| ------- | -------------------------------------------------------------------------------------------------------- |
| context | Webpack 在寻找相对路径的文件时会以 context 为根目录，context 默认为执行启动 Webpack 时所在的当前工作目录 |
| entry   | webpack 开始构建捆绑包的地方                                                                             |
