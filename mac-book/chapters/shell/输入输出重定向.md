# 输入输出重定向

- 可以操纵的流

| 流         | 值  |
| ---------- | --- |
| 标准输入流 | 0   |
| 标准输出流 | 1   |
| 标准错误流 | 2   |

- 重定向符

| 重定向符 | 含义             |
| -------- | ---------------- |
| <        | 输入重定向符     |
| >        | 覆盖输出重定向符 |
| >        | 追加输出重定向符 |

```shell
$ pwd
/tmp/demo
$ pwd >output_file
$ ls
output_file
$ cat output_file 
/tmp/demo
```
