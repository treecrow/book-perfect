# 系统快捷键

| 快捷键           | more                            |
| ---------------- | ------------------------------- |
| shift+⌘+g        | 在 Finder 中进入指定的目录      |
| ⌥+⌘+space        | 打开 Finder                     |
| ⌘+shift+.        | Finder 中隐藏与显示被隐藏的文件 |
| ⌘+⇧+esc          | 调出“强制退出应用”窗口        |
| 开机+Command + r | 进入恢复程序界面                |
