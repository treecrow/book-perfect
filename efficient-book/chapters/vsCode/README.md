# code 

> vsCode 命令行工具

| 命令        | more                                               |
| ----------- | -------------------------------------------------- |
| code .      | 打开当前路径所在文件                               |
| code folder | 打开文件夹(多个文件夹对应打开多个项目)             |
| code file   | 打开／创建 文件                                    |
| code -h     | 帮助                                               |
| code -v     | 查看 vsCode 版本号                                 |
| code -n     | 打开一个新的VS Code会话窗口                        |
| code -r xxx | 强制在上一个激活窗口中打开文件或文件夹(原项目不在) |
| code -g     | -                                                  |
| code -d     | 打开对比编辑器，需要两个文件路径作为参数           |
