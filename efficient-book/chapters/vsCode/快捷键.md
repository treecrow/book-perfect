# 快捷键

> `⌘K + ⌘R` 键盘快捷方式参考

## 符号按键对照表

| 符号 | 按键                                      |
| ---- | ----------------------------------------- |
| ⌘    | meta 按键(Mac--command、 Windows-windows) |
| ⇧    | shift                                     |
| ^    | ctrl                                      |
| ⌥    | alt                                       |
