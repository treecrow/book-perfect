# Spotlight

| 快捷方式                                 | 描述                     |
| ---------------------------------------- | ------------------------ |
| ⌘+space                                  | 启动                     |
| 回车                                     | 打开文件                 |
| 长按 ⌘                                   | 显示文件路径             |
| ⌘+回车 / ⌘+r                             | 打开文件所在文件夹       |
| kind:(file type/file format)             | 搜索指定类型/格式文件    |
| kind:(file type/file format) "file name" | 根据名称搜索指定类型文件 |
| ⌘+b                                      | 百度相关内容             |
