## 图片延迟加载并淡入淡出显示效果
```css
img {  opacity: 1;  transition: opacity 10s;  }
img[data-src] {  opacity: 0;  }
```
```js
[].forEach.call(document.querySelectorAll('img[data-src]'),function(img){
    img.setAttribute('src', img.getAttribute('data-src'));
    img.onload = function() {
        img.removeAttribute('data-src');
    };
});
```
