# 数组操作

```javascript
var arr = [
  { id: 1, name: "xiaowang", score: 22 },
  { id: 2, name: "xiaoli", score: 65 },
  { id: 3, name: "xiaozhang", score: 37 }
];
```

## forEach 循环改变原数组

```javascript
var arr = [1, 2, 3, 4, 5];
arr.forEach(function(item) {
  item = item + "~~~";
});
var arr2 = [1, 2, 3, 4, 5];
arr2.forEach(function(item, index, arr) {
  arr[index] = item + "~~~";
});
```
