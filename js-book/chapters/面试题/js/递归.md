# 递归

## 返回一个数的各个因子

```js
function getPreGens(num) {
  var arr = [];
  var i = 2;
  while (i < Math.floor(num / 2) + 1) {
    if (num % i == 0) {
      arr.push(i);
      console.log(num, i);
      num = num / i;
      i = 1;
    }
    i++;
  }
  if (num > 2) arr.push(num);
  return arr;
}
alert(getPreGens(180)); //2,2,3,3,5
```

## 求 1-100 的和

```js
function sum(n) {
  if (n == 1) return 1;
  return sum(n - 1) + n;
}
console.log(sum(100)); // 5050
```
