# 转义字符

## 转义字符列表

| 转义字符   | 含义                        |
|--------|---------------------------|
| \o     | NUL字符                     |
| \b     | 退格符                       |
| \t     | 水平制表符                     |
| \n     | 换行符                       |
| \v     | 垂直制表符                     |
| \f     | 换页符                       |
| \r     | 回车符                       |
| \"     | 双引号                       |
| \'     | 撇号或单引号                    |
| \\     | 反斜杠                       |
| \xXX   | 由两位16进制数XX指定的Latin-1字符    |
| \uXXXX | 由4位16位进制数XXXX指定的Unicode字符 |
