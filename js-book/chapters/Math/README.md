# Math

## 属性列表

属性           | 含义
------------ | --------------
Math.E       | 自然对数的底数,即常量e的值
Math.LN10    | 10的自然对数
Math.LN2     | 2的自然对数
Math.LOG2E   | 以2为底e的对数
Math.LOG10E  | 以10为底e的对数
Math.PI      | π值
Math.SQRT1_2 | 1/2的平方根
Math.SQRT2   | 2的平方根
