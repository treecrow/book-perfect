## isNaN

> 用于检查其参数是否是非数字值

- isNaN(null)---false:判断 null 之前转化为 0 了

## 判断是否为真

```javascript
function judgeTrue(val) {
  if (val === null) return false;
  if (!isNaN(val)) return val > 0;
  if (typeof val === 'object') return Object.keys(val).length === 0 ? false : true;
  return !!val;
}
```
