# 接口

## 接口(interface) 语法

| 点             | 语法                      | more                             |
| -------------- | ------------------------- | -------------------------------- |
| 可选属性       | ?                         | -                                |
| 只读属性       | readonly                  | -                                |
| 额外的属性检查 | [propName: string]: any;  | 默认情况下，传入多余的属性会报错 |
| 函数类型       | 略                        | -                                |
| 可索引的类型   | [index: T]: T;            | -                                |
| 类类型         | implements                | 比较复杂                         |
| 继承接口       | interface aaa extends bbb | -                                |
| 混合类型       | -                         | -                                |
| 接口继承类     | -                         | -                                |
