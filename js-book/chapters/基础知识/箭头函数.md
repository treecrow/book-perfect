# 箭头函数

* new 不可用。箭头函数不能使用 new 关键字来实例化对象，不然会报错。
* this 不可变。函数内置 this 不可变，在函数体内整个执行环境中为常量。
* 没有 arguments 对象。更不能通过 arguments 对象访问传入参数。只能使用显式命名或其他 ES6 新特性来完成。

## 箭头函数特点

| 特点      | more                                                                                       |
| --------- | ------------------------------------------------------------------------------------------ |
| 关联 this | 对 this 的关联。函数内置 this 的值，取决于箭头函数在哪儿定义，而非箭头函数执行的上下文环境 |
