## === 与 == 运算符

| 判断                  | ===   |
| --------------------- | ----- |
| 两个值不是相同类型    | false |
| obj1.obj2             | false |
| null,null             | true  |
| undefined,undefined   | true  |
| true,true/false,false | true  |
| NaN,__                | false |
| 0,0/-0,0              | true  |

- ===

> 如果两个值类型相同，按照===比较方法进行比较

| 判断                            | ==   |
| ------------------------------- | ---- |
| null,undefined                  | true |
| 3,"3"                           | true |
| true,1/false,0                  | true |
| function a(){},"function a(){}" | true |
