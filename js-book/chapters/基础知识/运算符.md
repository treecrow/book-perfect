# 运算符

| 运算符                                                         | 功能                                                                    |
| -------------------------------------------------------------- | ----------------------------------------------------------------------- |
| ++                                                             | -                                                                       |
| --                                                             | -                                                                       |
| +                                                              | -                                                                       |
| -                                                              | -                                                                       |
| \*、/、%                                                       | -                                                                       |
| =                                                              | -                                                                       |
| !                                                              | 逻辑非                                                                  |
| delete                                                         | 删除属性                                                                |
| typeof                                                         | 检测操作数类型                                                          |
| void                                                           | 返回 undefined 值                                                       |
| instanceof                                                     | 测试对象类型                                                            |
| in                                                             | 测试属性是否存在                                                        |
| ==                                                             | -                                                                       |
| ===                                                            | -                                                                       |
| !=                                                             | -                                                                       |
| !==                                                            | -                                                                       |
| &&                                                             | -                                                                       |
| `\\`                                                           | -                                                                       |
| ?:                                                             | -                                                                       |
| ~                                                              | 按位求反                                                                |
| <<                                                             | 左移位                                                                  |
| >>                                                             | 有符号右移                                                              |
| >>>                                                            | 无符号右移                                                              |
| &                                                              | 按位与                                                                  |
| ^                                                              | 按位异或                                                                |
| `\` | 按位或                                                   |
| \*=、/=、%=、+=、-=、&=、^=、`\`=、<<=、>>=、>>>= | 运算且赋值 |
| eval()                                                         | -                                                                       |
| \*\*                                                           | 指数运算符                                                              |
| ...                                                            | 将一个数组(包括类数组)转为用逗号分隔的参数序列 / 将字符串转为真正的数组 |
