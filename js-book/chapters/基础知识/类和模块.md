# 类和模块

## 检测任意对象的类

| 方式                       | 解释                   | 优缺点                         |
|--------------------------|----------------------|-----------------------------|
| instanceof 运算符           | -                    | 无法通过对象来获得类名，只能检测对象是否属于指定的类名 |
| obj1.isPrototypeOf(obj2) | 检测 obj1 是否是 obj2 的原型 | ~                           |
| constructor 属性           | -                    |                             |
| 构造函数的名字                  | -                    |                             |
