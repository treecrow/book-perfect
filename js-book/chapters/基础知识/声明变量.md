# 声明变量

## var、let、const 区别

| \#    | 声明次数     | 作用域                             | 变量提升         | 是否是顶层对象属性 |
| ----- | ------------ | ---------------------------------- | ---------------- | ------------------ |
| var   | 可重复声明   | 作用于是函数作用域                 | 存在变量提升现象 | 可以是             |
| let   | 只能声明一次 | 块级别作用域（大括号括起来的内容） | 不存在变量提升   | 不是               |
| const | 只能声明一次 | 块级别作用域（大括号括起来的内容） | 不存在变量提升   | 不是               |

### const（补充）

> 声明一个只读的常量。一旦声明，常量的值就不能改变。但是可以修改变量对应对象的属性
