# [File](https://www.electronjs.cn/docs/api/file-object)

> DOM 文件接口为原生文件提供了抽象, 以便让用户使用 HTML5 文件 API 直接处理原生文件。 Electron已经向 文件 接口添加了一个 path 属性, 在文件系统上暴露出文件的真实路径

## api 列表

| api       | more                             |
| --------- | -------------------------------- |
| file.path | 在文件系统上暴露出文件的真实路径 |
