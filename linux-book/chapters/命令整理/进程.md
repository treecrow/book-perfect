# 进程

| 命令                                      | more                                            |
| ----------------------------------------- | ----------------------------------------------- |
| [ps](http://man.linuxde.net/ps)           | 用于报告当前系统的进程状态 `ps -ax ／ grep xxx` |
| [killall](http://man.linuxde.net/killall) | 使用进程的名称来杀死进程(杀死一组同名进程)      |
| [kill](http://man.linuxde.net/kills)      | 用来删除执行中的程序或工作                      |
