# 身份

| 命令                                    | more                                       |
| --------------------------------------- | ------------------------------------------ |
| [who](http://man.linuxde.net/who)       | 显示目前登录系统的用户信息                 |
| [whoami](http://man.linuxde.net/whoami) | 打印当前有效的用户名称                     |
| [sudo](http://man.linuxde.net/sudo)     | 用来以其他身份来执行命令，预设的身份为root |
