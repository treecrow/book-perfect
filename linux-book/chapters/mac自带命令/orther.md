| ------------------------------------------------------------------ | ------------------------------------------------------------------------- |
| [rpm](http://man.linuxde.net/rpm)                                  | RPM 软件包的管理工具                                                      |

| [brew](https://brew.sh/)                                           | macOS 缺失的软件包的管理器                                                |
| [telnet](http://man.linuxde.net/telnet)                            | 用于登录远程主机，对远程主机进行管理                                      |
| [domainname](http://man.linuxde.net/domainname)                    | 用于显示和设置系统的 NIS 域名                                             |
