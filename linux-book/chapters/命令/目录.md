# 目录

| 命令                                  | more                                 |
| ------------------------------------- | ------------------------------------ |
| [pwd](http://man.linuxde.net/pwd)     | 以绝对路径的方式显示用户当前工作目录 |
| [cd](http://man.linuxde.net/cd)       | 切换工作目录                         |
| [ls](http://man.linuxde.net/ls)       | 显示目标列表                         |
| l(ls -la)                             | ~                                    |
| la(ls -A)                             | ~                                    |
| ll(ls -l)                             | ~                                    |
| [mkdir](http://man.linuxde.net/mkdir) | 创建目录                             |
