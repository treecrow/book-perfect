## npm 相关文件（和路径）

| file           | 功能                       |
| -------------- | -------------------------- |
| ~/.npmrc       | npm 配置文件               |
| ~/.npm-init.js | 自定义 package.json 键值对 |
| ~/.npm/`_logs` | 日志存储位置               |
