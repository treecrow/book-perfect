# npm scripts

## 执行顺序

| 命令 | more                                 |
| ---- | ------------------------------------ |
| &    | 并行执行                             |
| &&   | 链式任务（一个任务接着一个任务运行） |
| <    | 将文件内容输入到一个命令             |
| >    | 将命令输出重定向到文件               |
| \    | 将一个命令的输出重定向到另一个命令   |

## 配置环境

```json
“dev-mac”: " (export) NODE_ENV=development && xxx",
“dev-win”: " set NODE_ENV=development && xxx",
```
