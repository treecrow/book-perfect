# 语义化版本

> 语义版本号分为 X.Y.Z 三位，分别代表主版本号、次版本号和补丁版本号。当代码变更时，版本号按以下原则更新

| 原则 | more                                            |
| ---- | ----------------------------------------------- |
| -    | 如果有大变动，向下不兼容，需要更新 X 位         |
| -    | 如果是新增了功能，但是向下兼容，需要更新 Y 位。 |
| -    | 如果只是修复 bug，需要更新 Z 位。               |
