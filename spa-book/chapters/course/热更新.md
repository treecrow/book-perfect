# 热更新

- 安装依赖

```bash
npm i -D webpack-dev-server
```

- 相关配置

```js
// package.json
{
  "scripts": {
    "serve": "NODE_ENV=development webpack-dev-server --config ./.config_webpack/dev.js"
  }
}

// webpack
{
  plugins: [
    // 模块热替换
    new webpack.HotModuleReplacementPlugin()
  ],

  // ========== devServer ==========
  devServer: {
    publicPath: "",
    hot: true,
    port: 3072,
    open: true
  }
};
```
