# 提取 node_modules 模块

- 相关配置

```js
// webpack
{
  optimization: {
    // 代码分割
    splitChunks: {
      chunks: "all",
      cacheGroups: {
        // 提取所有 node_modules 的模块
        vendors: {
          name: `vendors`,
          test: /[\\/]node_modules[\\/]/,
          priority: 15,
          chunks: "initial",
          reuseExistingChunk: true
        }
      }
    }
  }
}
```
