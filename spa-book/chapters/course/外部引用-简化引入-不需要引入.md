# 外部引用-简化引入-不需要引入

- 相关配置

```js
// webpack
{
  // 外部引用（不需要安装，直接使用）
  externals: {
    jquery: "$"
  },
  resolve: {
    // 简化引入（引入路径简化）
    alias: {
      "@": path.resolve(__dirname, "../app")
    },
    // 省略文件相关扩展名
    extensions: [".js", ".vue", ".json"],
  },
  plugins: [
    // 简化引入(不需要引入，安装后直接使用)
    new webpack.ProvidePlugin({
      Vue: ["vue/dist/vue.esm.js", "default"]
    })
  ]
}
```
