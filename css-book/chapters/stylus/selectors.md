# selectors

| selector   | more                                             |
| ---------- | ------------------------------------------------ |
| 层层包裹   | -                                                |
| 逗号分割   | 换行作用一样                                     |
| ^[N]       | 选择当前块层级中的第N级选择器，N为负数则倒序计算 |
| ^[N..M]    | 选择当前块层级中的第N到M级选择器                 |
| &          | 当前选择器的父级选择器                           |
| ~/         | 当前块的第一个层级选择器                         |
| ../        | 当前选择器的爷爷级选择器，相当于 & 的 &          |
| ../../     | -                                                |
| /          | 相当于根选择器，此选择器祖先选择器失效           |
| selector() | 内置的编译器，解析出当前父级选择器真实的样貌     |
