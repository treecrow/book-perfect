# operators

## 运算符优先级

| 运算符                 | more               |
| ---------------------- | ------------------ |
| .                      | -                  |
| []                     | 数组索引           |
| ! ~ + -                | 一元运算符         |
| ** * / %               | 次方、乘、除、整除 |
| + -                    | -                  |
| ... ..                 | 序列               |
| <= >= < >              | 相等关系           |
| == is != is not isnt   | ~                  |
| in                     | 是否存在           |
| is a                   | 判断               |
| is defined             | ~                  |
| && and `||` or         | 逻辑操作           |
| ?:                     | -                  |
| = := ?= += -= *= /= %= | -                  |
| not                    | -                  |
| if unless              | -                  |
