# 居中

```css
/*父元素的宽高需要有设置*/
.zh-transform-center{display:block; position: relative; left: 50%;  top: 50%; transform: translate(-50%,-50%);}
/*该元素的宽高需要有设置,子元素需要时inline*类型的*/
.zh-cell-center{display:table-cell; vertical-align:middle; text-align:center;}
/*该元素的父元素需要设置position:relative*/
.zh-position-center{position:absolute;top:0;right:0;bottom:0;left:0;margin:auto;}
```
