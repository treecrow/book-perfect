常用命令

常用命令

git status命令可以让我们时刻掌握仓库当前的状态

$ git status

查看某个文件具体修改了什么内容

$ git diff readme.txt

显示从最近到最远的提交日志

$ git log
//简化的提交日志
$ git log --pretty=oneline

Git提供了一个命令git reflog用来记录你的每一次命令

$ git reflog
