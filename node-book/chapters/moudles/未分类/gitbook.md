## 安装

```
npm install -g gitbook
npm install -g gitbook-cli
```

## 命令列表

命令                           | 含义
---------------------------- | ----------------
gitbook init                 | 初始化一个gitbook项目
gitbook install              | 安装gitbook项目声明到插件
gitbook serve --lrport 33333 | 在33333端口监控进程
--port                       | web 服务监听端口
gitbook build                | -
