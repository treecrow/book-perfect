模块

模块 功能
chai 断言库，可以与任何测试框架配合，包括expect(), assert() 和 should三种风格
cheerio 后端的jquery,爬虫专用
co The ultimate generator based flow-control goodness for nodejs
CodeMirror 浏览器内嵌代码编辑器
debug 用于程序的调试（特殊的console?）
delegates Nodejs method and accessor delegation utility
escape-html Escape string for use in HTML
express 后台框架
highlight.js Javascript语法高亮工具
ionic 移动端混合开发框架
is-generator-function 判断一个函数是否是ES6 generator函数
istanbul JavaScript 程序的代码覆盖率工具
jquery jQuery JavaScript库
koa 轻量化的后台框架
loopback 基于express的后台框架
marked markdown解析编译器
matcha A caffeine driven, simplistic approach to benchmarking(性能测试？)
mocha 测试框架
node-formidable 解析表单数据，尤其针对文件的上传（视频，图片等等）
node-geohash geohash library for nodejs(根据ip地址获取最近的8个方向的地理位置)
node-only return whitelisted properties of an object
depd Deprecate all the things(特殊的console)
react 前端框架
should.js BDD风格的断言库（should）
standard JavaScript 代码规范，自带 linter & 代码自动修正
supertest node.js HTTP servers 断言库(基于superagent)
superagent 轻量化渐进的的客户端HTTP请求库
vue 前端框架
