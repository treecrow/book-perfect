未分类的模块

模块 功能
delegates Nodejs method and accessor delegation utility(整合一个对象的方法和属性到另外一个对象上)
escape-html Escape string for use in HTML
is-generator-function 判断一个函数是否是ES6 generator函数
matcha A caffeine driven, simplistic approach to benchmarking(性能测试？)
node-formidable 解析表单数据，尤其针对文件的上传（视频，图片等等）
node-geohash geohash library for nodejs(根据ip地址获取最近的8个方向的地理位置)
node-only return whitelisted properties of an object
nodejs-depd Deprecate all the things(特殊的console)
opener 打开资源，如网页、文件等
rimraf A rm -rf util for nodejs
standard JavaScript 代码规范，自带 linter & 代码自动修正
