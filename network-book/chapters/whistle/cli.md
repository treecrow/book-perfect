## 命令列表

| 命令               | more                                                        |
| ------------------ | ----------------------------------------------------------- |
| start              | 开启一个后台服务                                            |
| stop               | 关闭当前的后台服务                                          |
| restart            | 重启当前的后台服务                                          |
| run                | 开启一个前台服务                                            |
| help               | 展示帮助文档                                                |
| status             | 展示当前 whistle 的运行状态                                 |
| ===                | ===                                                         |
| use/add [filepath] | Set rules from a specified js file (.whistle.js by default) |
