# 元组

> 与列表类似，不同之处在于元组的元素不能修改

```python
tuple = ( 'abcd', 786, 2.23, 'runoob', 70.2  )
tup1 = (50,)
```

## 元组脚本操作符

| 表达式                         | 结果                         | 描述         |
| ------------------------------ | ---------------------------- | ------------ |
| (1, 2, 3) + (4, 5, 6)          | (1, 2, 3, 4, 5, 6)           | 连接         |
| ('Hi!',) * 4                   | ('Hi!', 'Hi!', 'Hi!', 'Hi!') | 复制         |
| 3 in (1, 2, 3)                 | True                         | 元素是否存在 |
| for x in (1, 2, 3): print (x,) | 1 2 3                        | 迭代         |

## 内置函数

| 方法及描述 | 实例                   |
| ---------- | ---------------------- |
| len(tuple) | 计算元组元素个数。     |
| max(tuple) | 返回元组中元素最大值。 |
| min(tuple) | 返回元组中元素最小值。 |
| tuple(seq) | 将列表转换为元组。     |
