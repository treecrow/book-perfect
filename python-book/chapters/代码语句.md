# 代码语句

| 分类 | 语句     | more                                          |
| ---- | -------- | --------------------------------------------- |
| 函数 | def      | 定义函数                                      |
| ~    | lambda   | 定义匿名函数                                  |
| ~    | return   | 返回函数结果                                  |
| 变量 | global   | 在函数内定义或使用全局作用域变量              |
| ~    | nonlocal | 用来在函数或其他作用域中使用外层(非全局)变量  |
| ===  | ===      | ===                                           |
| ~    | del      | 删除                                          |
| ~    | yield    | 在 Python 中，使用了 yield 的函数被称为生成器 |
