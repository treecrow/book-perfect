# 代码语句

| 分类   | 语句     | more                                          |
| ------ | -------- | --------------------------------------------- |
| 函数   | def      | 定义函数                                      |
| ~      | lambda   | 定义匿名函数                                  |
| ~      | return   | 返回函数结果                                  |
| 变量   | global   | 在函数内定义或使用全局作用域变量              |
| ~      | nonlocal | 用来在函数或其他作用域中使用外层(非全局)变量  |
| 逻辑   | True     | 逻辑真                                        |
| ~      | False    | 逻辑假                                        |
| ~      | not      | 逻辑反                                        |
| 生成器 | yield    | 在 Python 中，使用了 yield 的函数被称为生成器 |
| ===    | ===      | ===                                           |
| ~      | del      | 删除                                          |
| ~      | import   | 导入                                          |
| ~      | None     | 相当于 null                                   |
| ~      | pass     | 略过                                          |
| ~      | class    | 类                                            |
| ~      | while    | -                                             |
| ~      | assert   | -                                             |
| ~      | async    | -                                             |
| ~      | await    | -                                             |
| ~      | break    | -                                             |
| ~      | continue | -                                             |
| ~      | for      | -                                             |
| ~      | from     | -                                             |
| ~      | if       | -                                             |
| ~      | raise    | -                                             |
| ~      | try      | -                                             |
| ~      | with     | -                                             |
