| ------------------ | --------------------------------- |
| abc.ABC            | 一个使用 ABCMeta 作为元类的工具类 |
| abc.ABCMeta        | 用于定义抽象基类（ABC）的元类     |
| abc.abstractmethod | 用于声明抽象方法的装饰器          |
