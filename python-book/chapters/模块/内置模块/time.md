# [time](https://docs.python.org/zh-cn/3/library/time.html)

> 该模块提供了各种时间相关的函数

## api list

| api              | more                                    |
| ---------------- | --------------------------------------- |
| time.sleep(secs) | 暂停执行调用线程达到给定的秒数          |
| time.time()      | 返回浮点数的 seconds since the epoch 。 |
