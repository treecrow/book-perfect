# [json](https://docs.python.org/zh-cn/3/library/json.html)

> json 相关

## api list

| api          | more                                       |
| ------------ | ------------------------------------------ |
| json.dump()  | 将 Python 对象按照 JSON 格式序列化到文件中 |
| json.dumps() | 将 Python 对象处理成 JSON 格式的字符串     |
| json.load()  | 将文件中的 JSON 数据反序列化成对象         |
| json.loads() | 将字符串的内容反序列化成 Python 对象       |
