# 列表

## api 列表

| class          | method                  | more                           |
| -------------- | ----------------------- | ------------------------------ |
| 添加、移除元素 | list.append(obj)        | 在列表末尾添加新的对象         |
| ~              | list.insert(index, obj) | 将对象插入列表                 |
| ~              | list.remove(obj)        | 移除列表中某个值的第一个匹配项 |
| ~              | list.clear()            | 清空列表                       |
| ===            | ===                     | ===                            |
| ~              | len(list)               | 列表元素个数                   |
| ~              | sorted(list)            | 对所有可迭代的对象进行排序操作 |

## 直接操作列表

- `+ && *`操作

  ```python {cmd=true}
  list_ = [1, 2, 3]
  list_ *= 2
  print(list_)
  list_ += [1000, 2000]
  print(list_)
  ```

- 列表切片

  ```python {cmd=true}
  list_ = [1, 2, 3, 4, 5]
  print(list_[1:4])
  print(list_[:])
  print(list_[::-1])
  ```

- 利用列表的生成式语法来创建列表

  ```python {cmd=true}
  f = [x for x in range(1, 10)]
  print(f,'\n')
  f = [x + y for x in 'ABCDE' for y in '1234567']
  print(f,'\n')
  f = [x ** 2 for x in range(1, 100)]
  print(f,'\n')
  f = (x ** 2 for x in range(1, 100)) # 生成器对象(不是列表)
  print(f,'\n')
  ```
