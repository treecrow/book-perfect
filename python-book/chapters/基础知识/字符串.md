## 直接操作字符串

- 取出指定位置的字符

  ```python {cmd=true}
  str_ = 'abc123456'
  print(str_[3])
  ```

- 字符串切片(从指定的开始索引到指定的结束索引)

  ```python {cmd=true}
  str_ = 'abc123456'
  print(str_[2:5])  # c12
  print(str_[2:])  # c123456
  print(str_[2::2])  # c246
  print(str_[::2])  # ac246
  print(str_[::-1])  # 654321cba
  print(str_[-3:-1])  # 45
  ```
