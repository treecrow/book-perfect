# 集合

## api 列表

| api           | more                                                                                 |
| ------------- | ------------------------------------------------------------------------------------ |
| set()         | 创建一个无序不重复元素集，可进行关系测试，删除重复数据，还可以计算交集、差集、并集等 |
| set.add()     | 为集合添加元素                                                                       |
| set.update()  | 给集合添加元素                                                                       |
| set.discard() | 删除集合中指定的元素                                                                 |
| set.remove()  | 移除指定元素(如果不存在会引发 KeyError)                                              |
| set.pop()     | 随机删除集合中的一个元素,并返回被删除的元素                                          |

## 交集、并集、补集、余集、子集和超集

```python {cmd}
set1 = {1, 2, 3}
set2 = {3, 4, 5}
set3 = {1, 2, 3, 4, 5}
print(set1 & set2)
print(set1 | set2)
print(set1 ^ set2)
print(set1 - set2)
print(set1 <= set3)
print(set3 <= set1)
```
